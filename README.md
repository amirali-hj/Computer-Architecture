# Computer Architecture & Processor Design

This repository contains RTL designs, architectural implementations, and coursework for advanced Computer Architecture. The projects traverse the complete design space from fundamental digital arithmetic circuits to complex pipelined processor microarchitectures, with a strong focus on **MIPS** and **RISC-V** Instruction Set Architectures.

## 📚 Course Modules & Implementations

### Part 1: Digital Arithmetic & Hardware Fundamentals
* **Numbering Systems:** Core concepts of digital data representation.
* **Arithmetic Logic:** Implementation of Adders, BCD Adders, and basic Multipliers.
* **Advanced Arithmetic:** Optimized Multipliers, Dividers, and Floating-Point arithmetic modules.
* **HDL Integration:** Hardware description and modeling using Verilog.

### Part 2: Instruction Set Architecture (ISA) & Datapath
* **ISA Fundamentals:** Foundational concepts of Instruction Set Architectures.
* **MIPS Architecture:** MIPS ISA design and comprehensive processor commands.
* **RISC-V Architecture:** Custom integration and exploration of the RISC-V instruction set.
* **Datapath & Control:** Foundational Datapath routing and Controller logic design.

### Part 3: Processor Microarchitecture
* **Single-Cycle Processors:** Complete structural design of the MIPS processor.
* **Multi-Cycle Processors:** Advanced multi-cycle MIPS implementation and microarchitecture.
* **Microprogramming:** Designing robust control units utilizing microprogramming techniques.

### Part 4: Advanced Architecture & Pipelining
* **Performance Evaluation:** Benchmarking and architectural performance analysis.
* **Pipelining:** Multi-stage pipeline design, hazard detection, and data forwarding (Pipeline 1 & 2).
* **Exception Handling:** Robust exception and interrupt handling within the processor datapath.

### Part 5: Memory & I/O Systems
* **Memory Hierarchy:** Design and analysis of cache systems and memory hierarchy structures.
* **Input/Output (I/O):** Interfacing and I/O system management.
